# User Management CRUD

A simple CRUD web application built with HTML, CSS and JavaScript for registering, editing, searching and deleting users.

Features:
- Create new users
- Edit existing users
- Delete users with confirmation
- Search users by name, email, role or age
- Validate required fields
- Prevent duplicate email registration
- Show success and error messages
- Handle empty states dynamically
- Persist data using Local Storage

Tecnologies :
- HTML
- CSS
- JavaScript

Purpose:
This project was developed to practise DOM manipulation, form handling, validation logic, state management and CRUD operations on the front end.
